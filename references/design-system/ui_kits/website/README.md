# UI Kit — Finca La Hermosa Website

Recreation of `fincalahermosa.onhercules.app`

## Pages
- **Inicio** — Hero, spaces strip, intro section, services grid, quote band, CTA
- **Experiencias** — Hero, category tabs (5), package cards (3-col), extras grid, CTA
- **Espacios** — Hero, 6 space cards with images
- **Nosotros** — Hero, story split, differentiators, location, dark CTA
- **FAQ** — Accordion with 8 questions

## Components
- `Navbar` — Fixed, transparent → frosted glass on scroll
- `Footer` — Dark verde, 4-column layout
- `CTASection` — Full-bleed photo + frosted glass card overlay
- `SectionOverline` — Terracota uppercase label pattern

## Files
- `index.html` — Entry point, loads React + Babel
- `SharedComponents.jsx` — Navbar, Footer, CTASection, SectionOverline
- `Pages.jsx` — All 5 page components

## Notes
- Images use Unsplash placeholders. Replace with real Finca La Hermosa photography in production.
- Against font loaded from `../../fonts/against-regular.otf`
- Cormorant Garamond + Poppins from Google Fonts CDN
